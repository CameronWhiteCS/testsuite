About

There are a lot of duplicate issues about the official test suite being down. There has been an effort started by its author on the SocialHub about trying to resurrect the test suite, but unfortunately not much progress has been made, and folks have tried! As an aside, it is worth joining the SocialHub. Back on topic: this test suite attempts to supplement the original should it make its way back online.

This suite is built using the go-fed library in order to create instanced and short-lived ActivityPub actors for test runs. The tests are roughly equivalent ports from the official test suite. However, the official test suite relied on a lot of questionnaire-style prompts for the S2S federation tests. While this unofficial test suite cannot eliminate all of them, a solid number have been replaced with machine-assisted tests.

This is a community resource, please use it kindly.
